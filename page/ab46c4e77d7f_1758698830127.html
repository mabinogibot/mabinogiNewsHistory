新瑪奇Mabinogi | 煉獄之扉

煉獄之扉需要4人組隊進入，每兩周會開放新的地下城

。9/18維護後 : 復活的幻象

。10/2維護後 : 菲斯匹亞德

。10/16維護後 : 覺醒的深海主君

。10/30維護後 : 無盡的噩夢
 

通關後會根據設置的階段不同，隨機獲得1-2個不同階段的煉獄之扉封印證明石，也可以在NPC商店中購買

 

。階段 0-5 : 煉獄之扉封印的第1階段證明石

。階段 6-10 : 煉獄之扉封印的第2階段證明石

。階段 11-15 : 煉獄之扉封印的第3階段證明石

。階段 16-20 : 煉獄之扉封印的第4階段證明石

。階段 21-25 : 煉獄之扉封印的第5階段證明石

↬ 煉獄之扉調查(初級) ↫
通關煉獄之扉4種任務
。復活的幻象(不限階段) ▶完成獎勵 : 煉獄之扉遺留物1個
。菲斯匹亞德(不限階段) ▶完成獎勵 : 煉獄之扉遺留物1個
。覺醒的深海主君(不限階段)▶完成獎勵 : 煉獄之扉遺留物1個
。無盡的噩夢(不限階段)▶完成獎勵 : 煉獄之扉遺留物1個
任務完成獎勵 : 煉獄之扉遺留物5個
↬ 煉獄之扉調查(中級) ↫
通關煉獄之扉4種任務的所有20階段
。復活的幻象(20階段以上)
。菲斯匹亞德(20階段以上)
。覺醒的深海主君(20階段以上)
。無盡的噩夢(20階段以上)
任務完成獎勵 : 煉獄冒險家戒指選擇箱子
↬ 煉獄之扉調查(高級) ↫
通關煉獄之扉4種任務的所有25階段
。通關1種(25階段) ▶完成獎勵 : 暗月之徵第二稱號取得券
。通關2種(25階段) ▶完成獎勵 : 影繞黑月第二稱號取得券
。通關3種(25階段) ▶完成獎勵 : 月冉暮光第二稱號取得券
。通關4種(25階段) ▶完成獎勵 : 黑月顯現第二稱號取得券
任務完成獎勵 : 北斗七星鐮刀外型卷軸

【每周調查結算獎勵】

每週四上午7點結算上週通關階段獎勵。

。階段20以下 : 獲得完成對應階段數量的煉獄之扉遺留物

。階段20以上 : 最多獲得煉獄之扉遺留物20個
 

*與NPC煉獄之扉調查官對話後可獲得獎勵

煉獄之扉遺留物
